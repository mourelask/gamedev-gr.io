- [Godot](https://godotengine.org/) (gdscript / c# / visual scripting)
- [GameMaker](https://gamemaker.io/en/gamemaker) (gml / visual scripting)

- [Lumberyard](https://aws.amazon.com/lumberyard/) (c++ / lua)
